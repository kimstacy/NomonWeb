Welcome to the Keyboard Study!

Please enter your user ID below using the Row Column Scanning Interface
Supported switch events: spacebar keypress and left mouse click

How to use the Row Column Scanning Interface:

(1) Wait until the row containing the item you want to select is highlighted in Dark Blue.

(2) At this time, activate your switch to select your desired row. The Interface will begin scanning through each column in the selected row.

(3) Wait until the column containing the item you want to select is highlighted in Purple.

(4) At this time, activate your switch to select the item.

(5) The interface will begin to scan through the rows as before. Repeat this process to select more items.

(6) If you've accidentally selected the wrong row, you can wait for two column scans to undo.
